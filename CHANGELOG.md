# Changelog

## [2.2.0] — 2026-03-18

### Навигация и UX

- Авторизованные пользователи теперь могут вернуться на главную страницу сайта
- На главной отображается кнопка «Личный кабинет» вместо «Войти / Регистрация» для авторизованных
- Добавлена ссылка «Главная» в футер личного кабинета

### Пинг серверов

- Добавлено отображение пинга (задержки) для каждого прокси-сервера в дашборде
- Пинг отображается в детальной карточке прокси с цветовой индикацией (зелёный/жёлтый/красный)
- Новый API-эндпоинт `/proxies/:orderId/ping` для измерения задержки до сервера

### Подтверждение заказа

- Добавлено модальное окно подтверждения перед оплатой на странице тарифов
- В окне отображается сводка: тариф, период, локация, количество устройств и итоговая сумма
- Двухшаговый процесс: «Оформить заказ» → подтверждение → «Перейти к оплате»

## [0.0.1] — 2025-01-01

### Первый релиз 🚀

- **Клиентский сайт** — React + Tailwind CSS, тёмная тема
  - Лендинг с описанием тарифов и FAQ
  - Регистрация / вход по Email и Telegram
  - Подтверждение Email при регистрации
  - Привязка Email для Telegram-аккаунтов
  - Восстановление пароля по email
  - Личный кабинет: дашборд, прокси, платежи, профиль
  - QR-код и tg:// ссылка для подключения прокси
  - График подключений за 24 часа (Chart.js)
  - Автопродление — переключатель для каждого заказа
  - Всплывающее окно changelog при обновлениях

- **Оплата** — YooKassa API v3
  - Создание платежей через redirect-flow
  - Webhook обработка (payment.succeeded / canceled)
  - Привязка IP-адресов YooKassa для безопасности
  - Автоактивация заказа после успешной оплаты

- **Каталог** — тарифы + выбор локации
  - Готовые тарифные планы (управляются из панели)
  - Выбор локации вручную (по флагу) или авто-балансировка
  - Автонастройка прокси на наименее загруженном сервере

- **Панель администратора** — расширение
  - CRUD тарифов
  - CRUD changelog
  - Просмотр клиентов и их заказов
  - Бан / активация клиентов
  - Просмотр всех заказов

- **Email уведомления** — nodemailer + SMTP
  - Письмо подтверждения email
  - Письмо восстановления пароля
  - Чек об оплате
  - Письмо о скором истечении подписки

- **Развёртывание**
  - Docker multi-stage build (backend + client)
  - deploy.sh — интерактивная установка на VPS
  - Nginx + Let's Encrypt SSL
  - systemd автозапуск
